Battle API
A read-only API for cataloging battle rap stuff.

Endpoints

GET: /battler/:artist/:league

Description:
Returns information for an artist based on league.

Data:
{"battles": [{"video_id", "likes", "dislikes", "comments", "views", "description", "partners": [...], "published", "opponents": [...]}, ...], "info": {...}, "leagues": [...]}

Example:
https://battleapi.herokuapp.com/battler/loaded%20lux/all

GET: /battlers/:league

Description:
Returns list of artists for a specific league.

Data:
[{"battler", "battle_count", "views", "comments", "likes", "dislikes", "comments_per_battle", "views_per_battle", "likes_percentage", "view_percentage", "recent"}, ...]

Example:
https://battleapi.herokuapp.com/battlers/theurltv

GET: /battle/:id

Description:
Returns information for a battle based on its YouTube ID.

Data:
{"info": {"video_id", "battlers", "views", "likes", "dislikes", "description", "published", "channel", "likes_percentage"}}, "historical": [{...}, ...]}

Example:
https://battleapi.herokuapp.com/battle/6sJIvDq4UnQ

GET: /battles/:league

Description:
Returns battles for a league.

Data:
[{"title", "views", "likes", "dislikes", "comments", "channel", "published", "description"}, ...]

Example:
https://battleapi.herokuapp.com/battles/dontflop

GET: /leagues

Description:
Returns league information.

Data:
[{"short", "league", "views", "channel"}, ...]

Example:
https://battleapi.herokuapp.com/leagues
